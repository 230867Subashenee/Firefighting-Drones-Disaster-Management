# Firefighting Drones for Disaster Management

This project focuses on designing a drone-based system to assist emergency responders during fire and disaster situations.

# Features
- Real-time fire detection using sensor and thermal imaging concepts
- Remote monitoring of high-risk areas
- Improved situational awareness using AI and computer vision principles

# Technologies Used
- Python
- Computer Vision
- AI / ML Concepts
- Sensors and Drone Systems

# Objective
To reduce human risk and improve response efficiency during fire emergencies by enabling remote assessment and monitoring.
